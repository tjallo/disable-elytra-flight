# Disable Elytra Flight

A Minecraft mod that disables elytra flight everywhere except The End.

## Features

- Elytra flight works normally inside The End
- Elytra flight is blocked in all other dimensions (Overworld, Nether, etc.)
- Shows an action bar message when a player attempts to fly outside The End
- Message has a 2-second cooldown to avoid spam

## Compatibility

| Minecraft | Fabric | NeoForge |
|-----------|--------|----------|
| 1.21.1    | ✅     | ✅       |
| 1.21.11   | ✅     | ✅       |
| 26.2.x    | ✅     | ✅       |

## Installation

Download the JAR for your Minecraft version and mod loader from the [releases page](https://github.com/tjallo/disable-elytra-flight/releases) and place it in your `mods` folder.
